AppDNA SDK

Use the AppDNA SDK to embed AppDNA functionality within a familiar process or user interface, or to embed it within a simple webpage. Embedding AppDNA functionality into a process or user interface meshes the AppDNA tasks into users' usual workflows, avoiding the need for familiarization with the AppDNA user interface.

Citrix Receiver for HTML5 SDK

Use the Citrix Receiver for HTML5 SDK to customize delivery for Citrix hosted apps and desktops through your website. The SDK allows you to enhance support for HDX sessions and to build a rich app experience in your Enterprise portals with the Receiver for HTML5.
